# Calculator

This repository is for DB visit code challenge.

## Clone the project

```
$ git clone https://github.com/zan768616253/db-visit-challenge.git
```

## Install dependencies

```
$ cd db-visit-challenge
$ go mod tidy
```

## Build Project

```
$ go build cmd/main.go
```

## Run Project

```
$ ./main -u DBUSER -p DBPASSWORD
```
